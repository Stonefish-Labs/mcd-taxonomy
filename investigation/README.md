# Investigation Guidance Framework

Every finding generated by this taxonomy — whether a single POI or a full behavioral pattern — should be accompanied by structured investigation guidance. This guidance enables an agent (human or AI) to efficiently move from detection to determination: is this malicious, benign, or inconclusive?

## Guidance Template

Each finding's investigation guidance follows this structure:

```
FINDING: [POI or Behavioral Pattern ID]
CONFIDENCE: [Low | Medium | High] (informed by POI combination + contextual signals)

WHAT WAS DETECTED:
  Summary of the specific evidence found.

WHAT THIS SUGGESTS:
  The suspected intent or attack pattern.

VERIFY NEXT:
  Specific follow-up actions to confirm or deny the hypothesis.
  - Action 1
  - Action 2
  - ...

ESCALATION INDICATORS:
  Additional evidence that, if found, increases confidence that this is malicious.
  - Indicator 1
  - Indicator 2
  - ...

DE-ESCALATION INDICATORS:
  Evidence that, if found, suggests this is benign.
  - Indicator 1
  - Indicator 2
  - ...

RECOMMENDED DISPOSITION:
  [Investigate Further | Flag for Human Review | Auto-Block | Informational]
```

This structure is designed to be consumed by an automated agent as a decision tree: detect, assess confidence, investigate, escalate or de-escalate, and recommend disposition.
